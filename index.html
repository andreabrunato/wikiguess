WikiGuess

A vibe coding quiz game by Andrea "Bruno" Brunato

How does WikiGuess work?

This is a quiz game where each question is generated from a real Wikipedia article. The title of the page is hidden from the text, and your goal is to guess the correct article based on the snippet and clues provided. You can also buy the article's image for a small point penalty, and answering quickly will earn you more points!

About the project

Developed by Andrea 'Bruno' Brunato, web developer, with the help of GitHub Copilot and a vibe coding approach.
Source code available on GitHub.

Technologies used

Built with Vue.js, Tailwind CSS, Nuxt, and DaisyUI

Wikipedia API

This game uses the Wikipedia API to fetch data and images from articles.
